# hangman
Hangman
A console-based Hangman game written in Python. Guess the hidden word letter by letter before the hangman is fully drawn — 10 errors and it's game over!

Download & Play
Download the latest executable (no Python required) from the Releases page.

Features:

Over 100 possible words to guess
Visual hangman that builds step by step with each wrong guess
Up to 10 errors before you lose
Tracks both correct and incorrect guessed letters
Option to guess the full word at any point
Input validation (no invalid characters accepted)

How to Run Locally:
Requirements:

Python 3.x installed on your machine

Steps:

Clone or download this repository
Open a terminal in the project folder
Run the following command:

python Hangman.py

How to Play:

The game picks a random word and tells you how many letters it has
Enter one letter at a time to guess the word
If the letter is in the word, it will be revealed in its correct position(s)
If the letter is not in the word, the hangman grows and you lose an error
At any point you can try to guess the full word when prompted
You win by guessing all the letters or the full word before reaching 10 errors
You lose if you make 10 errors

Example
Welcome to Hangman!!

The word has 6 letters

You have 10 errors left
